Modulo 3

Submódulo Descripción
Submodulo 1:Sistemas de Información Bases de datos y SQL. Durante la capacitación, aprendimos los conceptos fundamentales de las bases de datos, entendiendo su importancia para almacenar, organizar y gestionar grandes cantidades de información. Exploramos la estructura de una base de datos relacional y cómo se conectan sus tablas mediante claves. También dimos los primeros pasos en el lenguaje SQL (Structured Query Language), realizando consultas básicas para buscar, insertar, modificar y eliminar datos en una base de datos, lo que nos permitió interactuar directamente con la información de forma eficiente y estructurada.
Submodulo 2: Programación C++ y Algoritmos. Durante la capacitación, aprendimos los fundamentos de la programación utilizando el lenguaje C++, comenzando con conceptos básicos como variables, operadores, estructuras condicionales y ciclos. A través de ejercicios prácticos, desarrollamos algoritmos para resolver problemas lógicos y matemáticos, fortaleciendo nuestras habilidades en el pensamiento computacional. Esto nos permitió entender cómo se estructura un programa y cómo usar C++ para automatizar tareas simples y complejas.
